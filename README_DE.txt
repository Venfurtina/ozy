PPL(A) Lernbereich – lokale Website
===================================

Start:
1. Python-Umgebung öffnen
2. pip install -r requirements.txt
3. python app.py
4. Browser: http://127.0.0.1:5000

Hinweise:
- Der Bereich /learn wurde auf dieselbe Grundoptik wie calendar.html / styles.css angepasst.
- Enthalten sind 9 Fächer, 18 Kapitel und 72 neu formulierte Trainingsfragen.
- Quellenlinks zeigen auf frei zugängliche EASA-/LBA-/Aircademy-Quellen.
- Lizenzierte Originalfragen aus Aircademy werden NICHT lokal gespiegelt; dafür kannst du später separat dein Dashboard verlinken.
- Notizen und Lernfortschritt werden lokal im Browser gespeichert.
